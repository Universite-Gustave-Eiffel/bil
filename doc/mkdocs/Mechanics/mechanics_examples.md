# Mechanics

Models for solid mechanics, from linear elasticity to advanced plasticity models including the Barcelona Basic Model (BBM) for unsaturated soils.

| Model | Description |
|-------|-------------|
| [Elast](Elast_model.md) | Linear elasticity |
| [Elasd](Elasd_model.md) | Elasticity with damage |
| [Plast](Plast_model.md) | Hardening elastoplasticity (Drucker-Prager) |
| [BBM](BBM_model.md) | Barcelona Basic Model (unsaturated plasticity, Cam-Clay) |
