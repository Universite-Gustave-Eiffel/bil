# Durability & Chemistry

Models for chemical degradation of concrete and cementitious materials: chloride ingress, carbonation, sulfate attack, and acid attack.

| Model | Description |
|-------|-------------|
| [Duracem](Duracem_model.md) | Coupled carbonation of cement paste (FVM) |
| [Chloricem](Chloricem_model.md) | Chloride ingress in cement paste (FVM) |
| [Sulfaco](Sulfaco_model.md) | Sulfate attack on concrete (FVM) |
| [Sulfuricem](Sulfuricem_model.md) | Biogenic sulfuric acid attack (FVM) |
| [Sulfaconew](Sulfaconew_model.md) | Sulfate attack (updated formulation) (FVM) |
