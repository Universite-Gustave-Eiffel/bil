# Poromechanics

Coupled hydro-mechanical models based on Biot's theory of poroelasticity, for consolidation problems and unsaturated poromechanics.

| Model | Description |
|-------|-------------|
| [M7](M7_model.md) | Unsaturated poroelasticity (Biot, consolidation) |
| [Poroplast](Poroplast_model.md) | Poroplasticity |
