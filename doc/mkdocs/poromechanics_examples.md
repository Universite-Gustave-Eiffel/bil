# Poromechanics

Coupled hydro-mechanical models based on Biot's theory of poroelasticity, for consolidation problems and unsaturated poromechanics.

| Model | Description |
|-------|-------------|
| [M7](M7_model.md) | Unsaturated poromechanics (Biot, 1D consolidation) |
| [M7-1](M7-1_model.md) | Unsaturated poromechanics (extended formulation) |
