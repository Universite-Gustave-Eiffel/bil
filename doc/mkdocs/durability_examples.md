# Durability & Chemistry

Models for chemical degradation of concrete and cementitious materials: chloride ingress, carbonation, sulfate attack, and acid attack.

| Model | Description |
|-------|-------------|
| [Fick](Fick_model.md) | Diffusion by Fick's second law |
| [Duracem](Duracem_model.md) | Coupled carbonation of cement paste |
| [Chloricem](Chloricem_model.md) | Chloride ingress in cement paste |
| [ChloricemSCM](ChloricemSCM_model.md) | Chloride ingress with supplementary cementitious materials |
| [ChloricemEla](ChloricemEla_model.md) | Chloride ingress with mechanical coupling |
| [ChloricemDLM](ChloricemDLM_model.md) | Chloride ingress with double-layer model |
| [Sulfaco](Sulfaco_model.md) | Sulfate attack on concrete |
| [Sulfuricem](Sulfuricem_model.md) | Biogenic sulfuric acid attack |
| [Sulfaconew](Sulfaconew_model.md) | Sulfate attack (updated formulation) |
| [SulfacoESA3d](SulfacoESA3d_model.md) | External sulfate attack (3D) |
