# Coupled Transfers

Models combining multiple transport phenomena: drying-wetting with salt crystallization, unsaturated soil behavior with coupled flows.

| Model | Description |
|-------|-------------|
| [DWS1](DWS1_model.md) | Drying-wetting with salt crystallization (FVM) |
| [Yuan1](Yuan1_model.md) | Coupled heat and moisture transfer (FVM) |
| [Shen](Shen_model.md) | Coupled flow in unsaturated soils (FVM) |
| [usoil](usoil_model.md) | Unsaturated soil (coupled hydro-mechanical) (FEM) |
