# Flow

Models for flow problems in porous media, including unsaturated flow (Richards equation), diffusion (Fick's second law), and two-phase flows.

| Model | Description |
|-------|-------------|
| [M1](M1_model.md) | unsaturated flow (FVM): 1d example |
| [M2](M2_model.md) | two-phase liquid-gas flow (FVM): 1d example |
| [Fick](Fick_model.md) | diffusion flow (FVM): 1d example |
| [Richards](Richards.md) | unsaturated flow (FEM): 2d example |
