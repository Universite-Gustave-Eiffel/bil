char*() String_RemoveComments ( char const *  src,
char *  dest 
)

Remove the comments from src and copy it to dest. The pointer dest should point to an allocated space large enough to accomodate the uncommented src. We can give the same pointer, i.e. dest = src, in that case the string src will be modified. Return the pointer dest.
