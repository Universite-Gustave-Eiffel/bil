Evaluate expression EXPR at compile time and ensure that it is fulfilled.

The expression must be given as the first argument. The second argument is a diagnostic string that should be part of the diagnostic message when the assertion fails.

Not all compilers give a useful message when an assertion is not fulfilled, but all should give at least the line number and stop compilation.

Remarks
This functionality will be directly supported in C11 and the macro static_assert must be provided by assert.h, which we include.

◆ max_align_t

typedef union max_align_t max_align_t

A type with the maximum alignment among the standard types.

This type is required for C11. We achieve this feature by simply producing a union of all the standard types, object and function pointer types, struct and union.

If it is provided, we additionally use a gcc extension to determine such a maximal alignment.

Remarks
The actual declaration of this type should never be used for any other purpose than to ensure alignment. The field names are obfuscated to make such abuses really difficult.
