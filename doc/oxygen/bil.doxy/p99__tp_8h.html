Declare a reference counter type for type T.

T should be a struct that contains a field

_Atomic(size_t) p99_cnt;

that is used as a reference counter. On initialization this counter must be set to 0.

In addition the following function must be provided:

void T ## _delete(T const*);

This function should take a pointer to an object of type T, destroy its contents (if necessary) and then call free on that pointer.

Warning
A type T that is administrated with that should never be allocated other than dynamically through malloc (or better P99_NEW) and friends.
See also
P99_TP_REF_FUNCTIONS for the macro that generates the inline functions that are needed for this to work
