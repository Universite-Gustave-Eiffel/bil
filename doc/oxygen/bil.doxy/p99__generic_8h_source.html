60 /* If no default is given, the idea of _Generic is to abort
61  compilation. To emulate this behavior we would have to produce an
62  expression that is valid as such, but can't be used in any valid
63  context. Such an expression doesn't exist, I think, since any valid
64  expression can always be used in a void context.
65 
66  The idea for that case is to use an extern function that never will
67  be defined and that bears a special attribute that inhibits this
68  function to be called. This function is then called through a
69  sizeof expression for VLA with side effect, such that it only
70  triggers when that particular branch is taken.
71 
72  Unfortunately gcc had this attribute only from about version 4.3,
73  so for versions below that this only produces a link error, much
74  later.
75  */
