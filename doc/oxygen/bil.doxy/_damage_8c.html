double Damage_UpdateTangentStiffnessTensor ( Damage_t *  damage,
double *  c 
)

Update the 4th rank tangent damage tensor in c. On input c should point to the elastic stiffness tensor. On output c is updated to the tangent damage stiffness tensor. Other inputs, included in damage, are: dfsde is the yield function gradient dgsde is the potential function gradient hm is the hardening modulus Tensor c is then updated as C(i,j,k,l) = C^el(i,j,k,l) - dgsde(i,j) * dfsde(k,l) / hm Return the inverse of hm: 1/hm
