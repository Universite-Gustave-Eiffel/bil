declare a thread local variable NAME of type T and with destructor DTOR.

See also
p99_tss
Remarks
such a variable must be declared in global scope

A hypothetical example for the use of such a variable would be errno:

P99_TSS_DECLARE_LOCAL(int, errno_loc);
#define errno P99_TSS_LOCAL(errno_loc)

With just these two lines errno always evaluates to an lvalue representing a thread local object. That is you can use it everywhere a normal variable of type int could be used:

if (errno == EINTR) ...
errno = 0;
my_func(&errno);

DTOR is optional and corresponds to the destructor function that is called when the thread finishes. If it is omitted, free is used. In that case you should better use P99_DECLARE_THREAD_LOCAL such that more efficient realizations than p99_tss could be used for the task.
