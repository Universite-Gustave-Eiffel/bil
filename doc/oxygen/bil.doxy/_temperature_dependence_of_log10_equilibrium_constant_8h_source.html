28 /*
29  * The Gibbs free energy change of reaction,
30  *
31  * DrG = SUM_i (nu_i * mu_i)
32  *
33  * is equal to 0 at equilibrium: DrG = 0. The ln(K) is linked to the Gibbs free
34  * energy change of the reaction in standard state, DrG0 = SUM_i (nu_i * mu0_i):
35  *
36  * DrG0 = - RT * ln(K)
37  *
38  * which itself is linked to the formation Gibbs free energy: DrG = SUM_i nu_i DfG(i).
39  *
40  * We compute DrG0 through the followings steps
41  *
42  * DrG(T) = DrH(T) - T * DrS(T)
43  * DrH(T) = DrH(T0) + Int_T0^T DrCp(T) dT
44  * DrS(T) = DrS(T0) + Int_T0^T DrCp(T)/T dT
45  *
46  * and so
47  *
48  * d/dT(R ln(K)) = -d/dT(DrG/T) = DrH/T^2 = DrH(T0)/T^2 + 1/T^2 Int_T0^T DrCp(T) dT
49  *
50  * DrG(T) = DrG(T0) - (T - T0) * DrS(T0) + (Int_T0^T DrCp(T) dT) - T * (Int_T0^T DrCp(T)/T dT)
51  *
52  * ie.
53  *
54  * R ln(K) = R ln(K0) - DrH(T0) * (1/T - 1/T0)
55  * - (1/T) * (Int_T0^T DrCp(T) dT) + (Int_T0^T DrCp(T)/T dT)
56  *
57  * 0. Seven-term temperature approximation: DrCp(T) = a0 + a1*T + a2/T^2 + a3*sqrt(T) + a4*T^2
58  * -------------------------------------------------------------------------------------------
59  * We find (in PHREEQC A6=A7=0)
60  *
61  * log(K) = A1 + A2*T + A3/T + A4*log(T) + A5/T^2 + A6*T^2 + A7*sqrt(T)
62  *
63  * 1. Five-term approximation: DrCp(T) = a0 + a1 * T + a2 / T^2 (adopted in PHREEQC and GEMS)
64  * ------------------------------------------------------------------------------------------
65  * [1] Maier, C.G., and Kelly, K.K., 1932,
66  * An equation for the representation of high temperature heat content data:
67  * Amer. Chem. Soc. Jour., v. 54, pp. 3243-3246.
68  *
69  * R ln(K) = R ln(K0) - DrH(T0) * (1/T - 1/T0) + a0 * ( T0/T - 1 + ln(T/T0) )
70  * + 1/2 * a1 * (T - T0)*(1 - T0/T) + 1/2 * a2 * (1/T - 1/T0)^2
71  *
72  * i.e.
73 
74  * log(K) = A1 + A2*T + A3/T + A4*log(T) + A5/T^2
75  *
76  *
77  * 2. Three-term temperature extrapolation: DrCP(T) = cst (adopted in CEMDATA)
78  * ---------------------------------------------------------------------------
79  * This corresponds with a constant heat capacity of reaction DrCp(T) = DrCp. Then
80  *
81  * DrG(T) = DrH(T) - T * DrS(T)
82  * DrH(T) = DrH(T0) + DrCp * (T - T0)
83  * DrS(T) = DrS(T0) + DrCp * ln(T/T0)
84  *
85  * R ln(K) = R ln(K0) - DrH(T0) * (1/T - 1/T0) + DrCp * ( T0/T - 1 + ln(T/T0) )
86  *
87  *
88  * So with the approximation (A2=A5=A6=A7=0):
89  * log(K) = A1 + A3 / T + A4 * log(T) = log(K0) + A3 * (1/T - 1/T0) + A4 * log(T/T0)
90  *
91  * A1 = + (1/(ln(10) * R)) * ( (DrS(T0)) - (DrCp) * ( 1 + ln(T0) )
92  * A3 = - (1/(ln(10) * R)) * ( (DrH(T0)) - (DrCp) * T0 )
93  * A4 = + (1/(R) * (DrCp)
94  */
