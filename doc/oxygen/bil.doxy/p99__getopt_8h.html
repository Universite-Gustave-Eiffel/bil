A list of constants that can be used for option processing.

Besides the characters that can be part of identifiers (alphanumeric characters and underscore), we provide several other characters for commandline option processing. The ones that are available can be deduced from the ending of the constant. E.g ::p99_getopt_enum_p00AT would be used with the token "AT" and corresponds to the @ character:

P99_GETOPT_DECLARE(AT, char const*, char_cptrVar, 0, 0, "just a string");
