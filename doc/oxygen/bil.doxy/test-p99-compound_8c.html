Detailed Description

Test the behavior of compound literals.

Compound literals may lead to the propagation of stack pointers outside there scope of validity. This little program tests the behavior of your compiler with respect to these, in particular if it detects the misbehavior.

All my compilers with just -Wall for the moment only detect myLessBuggyFunction, i.e an explicit temporary that is returned from a function.

Compound literal proliferation in myBuggyFunction is only detected by clang with the –analyze option.

valgrind detects this misshavior at run time
