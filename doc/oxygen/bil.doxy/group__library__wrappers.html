Detailed Description

Wrap library functions such that they throw an exception on error.

All macros here have the form of NAME_throw, where NAME is a C library function. All these macros are classified according to one of the conventions that are present in the C library on the strategies how errors are returned.

Eg. to have a malloc that always checks the return do

double (*A)[n] = malloc_throw(sizeof *A);

In case you want just blend the names of the system calls you could do yourself for the ones that you want something like

#define malloc malloc_throw
...
double (*A)[n] = malloc(sizeof *A);
Warning
Since these are just too many different calls only few of them have been tested. If you find a classification error please let us know.
