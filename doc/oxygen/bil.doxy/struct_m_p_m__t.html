◆ SetInputs

Values_d * SetInputs

Explanation of the parameters common to all functions. el = pointer to element object t = current time dt = time increment (i.e. the previous time is t-dt) u = pointer to pointer to primary nodal unknowns p = p^th interpolation Gauss point of the FE val = custom values at the current time val_n = custom values at the previous time (always an input) On output: val is initialized with the primary nodal unknowns (strain,pressure,temperature,etc...)

Return a pointer to val

◆ SetTangentMatrix

int SetTangentMatrix

On input: k = the k^th column of the tangent matrix to be filled. dval = the derivatives of val wrt the k^th primary unknown

On output: c = pointer to the matrix to be partially filled (only the column k).

Return the shift (the size of the matrix: ncols*nrows) if succeeds or < 0 if fails. Exemples:

  • for an elastic matrix, shift = 81
  • for a poroelastic matrix, shift = 100

◆ SetTransferMatrix

int SetTransferMatrix

On output: c = pointer to the transfer matrix to be filled

Return the shift (the size of the matrix: ncols*nrows) Example: if there are ndif diffusion process, shift = 9*ndif*ndif
