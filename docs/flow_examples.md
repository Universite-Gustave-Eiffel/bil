# Flow

Models for fluid flow in porous media, including unsaturated flow (Richards equation), diffusion (Fick's second law), and two-phase flows.

| Model | Description |
|-------|-------------|
| [M1](M1_model.md) | 1D unsaturated flow (Richards equation, FVM) |
| [M2](M2_model.md) | Two-phase liquid-gas flow |
| [M5](M5_model.md) | Water-air transfer |
| [M10-1](M10-1_model.md) | Heterogeneous 2D flow (variant 1) |
| [M10-2](M10-2_model.md) | Heterogeneous 2D flow (variant 2) |
| [Richards 2D](Richards-2d.md) | 2D unsaturated flow (FEM) |
